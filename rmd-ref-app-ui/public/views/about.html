About this RM&D Application

Remote monitoring and diagnostics (RM&D) applications are used in industries to monitor machines to report irregularities in operation and analyze patterns of behavior and performance over time. Monitoring and diagnostic applications use a set of algorithms, processes, and tools to monitor the health of an asset and detect faults before they turn into failures.

This RM&D reference app is a good starting point for understanding Predix cloud application development. This app was created from a fork of the Predix Experience Seed app, then the user interface was customized for the needs of RM&D. The app also demonstrates how you can

  • Integrate your backend services
  • Model assets and meters
  • Ingest data into time series
  • Customize your user interface
  • Secure your application using Predix authentication service
  • Use Predix Contextual Dashboard to browse assets
  • Use Predix widgets - Data Grid, and Time-series
  • Integrate Predix cloud services - Asset, Time Series, Dashboard Views
  • Visualize Predix Machine live data

    Using the application

    Click on Dashboard tab, and click "Select an Asset" to browse all assets. In a Predix dashboard application like this, views are associated to asset classifications. In this application there are views associated with groups of assets, like the Predix Energy > Richmond > Refinery group.
    To see aggregated details on the refinery group, select it in the asset browser, and click "Open". You will see a view with two components - a summary component at the top, and the datagrid at the bottom.
    To see details on an asset, browse to Predix Energy > Richmond > Refinery > Bently Compressor, and click "Open". You will see the "Compressor Data" view with several components: rmd-summary, rmd-px-data-table, ref-list-picker, ref-digital-meter, px-chart, etc. On this view, you can experiment with several of the UI components. In the "Asset Performance" datagrid, you can sort by clicking on column headers, search for meters, and refresh to see the latest meter readings. Below that, you can select multiple parameters to chart on the time series widget, and change the date range. In the top right corner is the "view selector", showing the current view as "Compressor Data". If you click on that you can switch to the "Live Data" view for the selected Bently Compressor asset.
    On the Live Data view, you'll see data streaming directly from Predix Machine to the browser. You can select different parameters, and stop/start the data stream.
